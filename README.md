# Next.js MCP Chatbot — Open-Source AI Assistant with Multi-LLM & MCP Support

A powerful, self-hostable AI chatbot built with **Next.js** and the **Vercel AI SDK**, 
featuring full **Model Context Protocol (MCP)** integration. Connect any LLM provider 
— OpenAI, Claude, Gemini, xAI, Ollama, or OpenRouter — and supercharge it with 
MCP tools, visual workflows, custom agents, realtime voice, web search, and 
image generation. Deploy free in minutes with Vercel or run locally with Docker.

> Inspired by ChatGPT, Claude, and Gemini — but open-source, local-first, and 
> infinitely extensible via MCP.
